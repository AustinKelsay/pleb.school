## Usage Notes

- **Aspect ratio:** All prompts use `--ar 16:9` for hero/header images. Change to `--ar 1:1` for thumbnails or `--ar 3:2` for cards.
- **Version:** Using `--v 6.1` for best quality. Adjust if using different Midjourney version.
- **Stylize:** `--s 150-250` range balances creativity with prompt adherence.
- **Variations:** After generating, use `--sref` with a favorite result to maintain style consistency across all images.
- **Upscale:** Use `U1-U4` on results, then `Upscale (Subtle)` for final high-res versions.
